# AWS Cloud IT Audit Project

## Overview
This project simulates a cloud IT audit engagement performed against an AWS environment.

The audit focused on evaluating the effectiveness of identity and access management, storage security, and logging controls within the environment.

The project demonstrates practical IT audit, GRC, and technology risk assessment skills using AWS cloud services.

---

## Audit Areas Reviewed

### Identity and Access Management (IAM)
- Privileged access review
- Least privilege assessment
- Multi-Factor Authentication (MFA)

### Storage Security
- S3 bucket access configuration
- Public access exposure risks

### Logging and Monitoring
- CloudTrail logging configuration
- Audit logging coverage

---

## Key Risks Identified
- Excessive privileged access
- Missing MFA controls
- Publicly accessible cloud storage
- Incomplete logging configuration
